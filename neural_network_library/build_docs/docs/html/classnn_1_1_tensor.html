Detailed Description

Dense n-dimensional tensor with optional gradient storage.

Tensor owns a flat contiguous buffer and a shape vector describing how that buffer should be interpreted. Most math operations return a new Tensor and, when gradients are enabled, attach lightweight autograd metadata so backward() can propagate gradients to upstream tensors.
